发布日期: 2026-05-27

此版本将 Ram Saver 内置进启动器，内存效率大大提升，基本上不会出现内存爆掉的情况了，这就意味着紫音、影之诗这种内存杀手也可以在大多数设备上流畅地运行了，也可以加更多模组游玩了，总体而言，提升非常大，建议升级到该版本。因为无法测试所有的模组和新内存优化的兼容性，如果遇到黑屏、卡顿严重、发热严重的问题，欢迎在反馈中提出！

## 新特性
- 新增内存优化，极大地提高了稳定性和帧数，将 Ram Saver 模组修改后加入启动器内置模组。
- 重写启动游戏时的加载屏幕，添加动画与美化。
- 更改市场评分图标，新增市场查看下载数。
- 设置-启动器底部添加一键重命名模组为模组原始名字的功能。
- 尽可能减少云存档弹窗弹出。
- 调整返回键返回启动器的弹窗。

## 修复
- 修复一些无法下载或安装导入的问题。
- 离线降采样导致的过度模糊问题。
