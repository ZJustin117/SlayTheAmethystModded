## 热修复
- 修复查看牌组或查看图鉴崩溃的问题。
- 修复指向性牌无法取消的问题。

## 新特性
- 新增混合触屏模式，默认启用，该模式下不仅保留了手机端原生触屏的特性，例如选牌二次确认，也让其他模组的 UI 保留电脑端样式。这就意味着之前所有因为点击导致的问题解决了，例如崩坠矛盾模组无法融合卡牌、岐路旅人模组无法转职的问题将不再需要切换到桌面输入模式来解决。
- 新增模组启动配置功能，可以保存和切换不同的模组启用组合。
- 主界面现在可以点击已启用模组的文字直接查看当前启用的模组。
- 提供了对繁体中文的支持。

## 修复
- 修复部分遗物获取界面在触屏操作下无法直接点选遗物的问题。
